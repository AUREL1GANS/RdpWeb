Apa perbedaan RDP dan Hosting biasa?

RDP (Remote Desktop Protocol) memberikan Anda akses penuh ke sebuah komputer virtual di cloud, dimana Anda bisa menginstall software, mengkonfigurasi sistem, dan memiliki pengalaman seperti menggunakan komputer fisik. Sedangkan hosting biasa biasanya terbatas pada layanan web hosting dengan kontrol yang lebih terbatas.

Bagaimana cara mengakses RDP setelah pembelian?

Setelah pembayaran diverifikasi, Anda akan menerima email yang berisi:

  • Alamat IP server
  • Username dan password
  • Port (jika tidak default)

Anda bisa menggunakan Remote Desktop Connection (Windows) atau aplikasi RDP client lainnya untuk terhubung.

Apakah ada garansi uptime?

Ya, kami memberikan garansi uptime 99.9%. Jika server mengalami downtime melebihi ini, Anda berhak mendapatkan kompensasi berupa perpanjangan masa aktif sesuai durasi downtime.

Bagaimana cara upgrade paket?

Anda bisa melakukan upgrade kapan saja melalui dashboard pelanggan. Sistem akan menghitung selisih harga dan masa aktif yang tersisa. Proses upgrade biasanya hanya membutuhkan beberapa menit tanpa downtime.

Apakah data saya akan di-backup?

Kami melakukan backup harian untuk semua paket hosting. Untuk paket tertentu, Anda juga bisa melakukan backup manual melalui control panel. Namun kami tetap menyarankan untuk melakukan backup mandiri secara berkala.
